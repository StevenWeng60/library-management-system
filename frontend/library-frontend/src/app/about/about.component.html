Welcome User!


Let me walk you through how this application works


Search for books

Search for books by utilizing the search function. There are also different ways to filter searches


Check books out

Check out books by utilizing the search bar. If there is at least one copy available, then you may check out the book


Check books in

Check in books by pressing the return button next to the book within your booklist


Recent additions

Check the recent additions tab to find a history of the most recent books added to the database


Additional rules

1. Each user can have at most, up to 10 books checked out

2. Each book can be checked out for up to 2 weeks


And thats it! Thanks for using this application
